IPHONE SETUP — GITHUB PAGES (KOSTENLOS, 5 MIN)

1

Gehe auf github.com → kostenloses Konto erstellen

2

Oben rechts „+" → „New repository" klicken

3

Name: water-check · Sichtbarkeit: Public · „Create repository"

4

„uploading an existing file" klicken → diese HTML-Datei hochladen → „Commit changes"

5

Gehe zu Settings → Pages → Branch: main auswählen → Save

6

Nach 1–2 Minuten läuft die App auf deinname.github.io/water-check

7

URL in Safari öffnen → Teilen → „Zum Home-Bildschirm" → fertig!
